# privacy-policy

Privacy policy for Top Shelf Tech and DrPine Solutions apps and Alexa skills.

## GitHub Pages

This repository is set up as a static GitHub Pages site:

- Main page: `index.html`
- Auto-deploy workflow: `.github/workflows/deploy-pages.yml`
- Deploy trigger: every push to `main`

### First-time enablement

In GitHub:

1. Go to `Settings` -> `Pages`.
2. Under `Build and deployment`, select `Source: GitHub Actions`.
3. Push to `main` (or re-run the workflow) and wait for deployment.

### Site URL

Default project site URL format:

`https://drpine.github.io/privacy-policy/`

If you set a custom domain later, GitHub Pages can serve this site there as well.
